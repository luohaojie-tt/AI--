# A股短线选股应用需求文档

## 1. 项目背景与目标

### 1.1 项目背景
- A股市场短线交易活跃，热门股波动大，存在较大的短线获利机会
- 普通投资者难以实时跟踪市场热点和龙头股
- 游资等专业短线选手的交易策略具有较高的参考价值
- 现有选股工具缺乏针对A股短线交易的专门设计

### 1.2 项目目标
- 开发一款基于游资短线交易策略的A股选股应用
- 实时监测市场热点和龙头股
- 基于情绪周期提供选股建议
- 辅助投资者制定短线交易决策
- 提高短线选股的效率和准确性

## 2. 核心短线策略

### 2.1 情绪周期分析
- **启动期**：新题材出现，市场情绪开始回暖
  - 特征：市场高度板连续三天在三板以下，突然出现一字跌停股翘板翻红
  - 策略：早盘快速介入首板股，试探市场反应

- **高潮期**：市场情绪高涨，热点板块持续强势
  - 特征：龙头股缩量加速，连板数量增加
  - 策略：接力前排龙头，避开后排跟风股

- **退潮期**：市场情绪开始降温，热点板块走弱
  - 特征：龙头股开始回调，连板数量减少
  - 策略：空仓或轻仓试错逆势活口

- **萧条期**：市场情绪低迷，赚钱效应差
  - 特征：高度板压制在低位，市场成交量萎缩
  - 策略：保持空仓，等待情绪拐点

### 2.2 打板选股策略
- **核心逻辑**：情绪博弈，紧跟市场最强资金动向
- **选股条件**：
  - 板块龙头或日内首板
  - 快速封板（封板时间早）
  - 封单量较大
  - 成交量放大
  - 换手率适中

### 2.3 龙头股识别
- **识别标准**：
  - 板块内最先涨停
  - 涨停次数最多
  - 成交量最大
  - 换手率适中
  - 市场关注度高
  - 抗跌性强

### 2.4 热点板块监测
- **监测指标**：
  - 板块涨幅排名
  - 板块内涨停股数量
  - 板块成交量变化
  - 板块资金流入情况
  - 板块持续时间

### 2.5 技术指标筛选
- **流动性指标**：日均成交额≥3亿元，防止进出困难
- **波动率指标**：20日年化波动率≥35%，有价差可做
- **趋势指标**：均线多头排列，处于上升趋势
- **量能指标**：成交量异常放大，换手率适中
- **涨跌幅指标**：近期强势上涨，涨幅居前

### 5.1 核心技术栈
- 后端：Python
- 前端：Tkinter（基于现有项目）
- 数据处理：Pandas、NumPy
- 可视化：Matplotlib
- 数据采集：Requests、API调用

### 5.2 系统架构
- 数据层：负责数据采集、存储和管理
- 策略层：实现各种短线交易策略
- 分析层：进行技术分析和情绪分析
- 可视化层：展示分析结果和选股推荐
- UI层：用户交互界面

### 5.3 性能要求
- 实时数据更新延迟 < 1秒
- 选股结果生成时间 < 5秒
- 图表渲染时间 < 1秒
- 支持同时监测所有A股股票
