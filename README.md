# SNR数据可视化工具 V1.0

## 项目概述

SNR数据可视化工具是一个专业的信噪比(Signal-to-Noise Ratio)数据分析和可视化应用程序。该工具提供了多种图表类型和交互功能，帮助用户深入分析SNR数据的分布特征和关联关系。

## 功能特性

### 📊 多样化图表支持
- **折线图**: 展示SNR随时间或序列的变化趋势
- **热力图**: 显示PRE/MAIN/POST参数的二维关联热力图
- **3D热力图**: 三维立体热力图，支持旋转和缩放交互
- **3D散点图**: 四维数据可视化(PRE/MAIN/POST/SNR)，支持精确的数据点交互
- **多子图**: 同时显示多个相关图表进行对比分析

### 🎯 交互功能
- **鼠标悬停**: 实时显示数据点详细信息
- **3D交互**: 支持旋转、缩放、平移操作
- **数据点拾取**: 精确的数据点选择和信息展示
- **视图切换**: 快速在不同图表类型间切换
- **缓存机制**: 智能缓存提升图表切换性能

### 📈 数据处理
- **CSV文件支持**: 支持标准CSV格式数据导入
- **数据验证**: 自动检测和验证数据格式
- **异步加载**: 大数据集异步处理，保持界面响应
- **统计分析**: 自动计算均值、标准差等统计指标

### 🎨 用户界面
- **现代化UI**: 基于Tkinter的现代化界面设计
- **中文支持**: 完整的中文界面和字体支持
- **状态反馈**: 实时状态栏信息和进度提示
- **响应式布局**: 自适应窗口大小调整

## 数据格式要求

### CSV文件格式
```csv
PRE,MAIN,POST,SNR
65386.0,100.0,65486.0,59.7
65386.0,150.0,65486.0,58.1
65386.0,200.0,65486.0,56.5
...
```

### 字段说明
- **PRE**: 前置参数值
- **MAIN**: 主参数值  
- **POST**: 后置参数值
- **SNR**: 信噪比值

## 安装和使用

### 环境要求
- Python 3.7+
- Windows 10/11 (推荐)

### 依赖安装
```bash
pip install -r requirements.txt
```

### 运行应用
```bash
python snr_visualizer_optimized.py
```

或使用批处理文件:
```bash
run.bat
```

### 使用步骤
1. 启动应用程序
2. 点击"加载数据"按钮选择CSV文件
3. 选择所需的图表类型
4. 使用鼠标交互查看数据详情
5. 切换不同视图进行对比分析
