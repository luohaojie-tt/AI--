## 建议的手动测试步骤

1. 启动程序，检查主窗口是否正常显示
2. 点击'加载数据'按钮，选择`data/sample_snr_data.csv`文件
3. 切换到折线图视图，检查图表是否正常显示
4. 切换到热力图视图，检查图表是否正常显示
5. 切换到3D热力图视图，检查3D图表是否正常显示
6. 切换到3D散点图视图，检查3D散点图是否正常显示
7. 使用筛选功能，设置参数范围进行筛选
8. 使用搜索功能，进行精确和模糊搜索
9. 测试鼠标交互功能，如悬停、旋转、缩放等
10. 检查数据点点击后是否能正确显示详细信息
